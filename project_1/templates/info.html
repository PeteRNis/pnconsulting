ABOUT


This application is built as an attempt to showcase my digital skills.


It is developed using an agile methodology and structured using Opbject Oriented Programming (OOP) and a Model View Controller (MVC) design pattern.


Python, HTML and CSS is the primary langueges used for the application, but Bootstrap classes has beed added to prettify the expirience.


The framework is written in flask, which integrates with a list of other libraries in order to ensure that the user can view and interact with the application


Within Python, Werkzeug has been used to authenticate, hash and secure the passwords of the users, and SQLAlchemy has been used as an Object-Relational Mapper ORM for automatic database transfer and storage.


The application is currently migrated with PostgreSQL, but it has been migrated with SQLite and MYSQL in the earlier stages of the project


The infrastructure is currently provided by Azure, but has been provided by Heroku in the past.
